## Event bookings website for Watermelon Studio

# Required settings

- SECRET_KEY
- DATABASE_URL
- EMAIL_HOST_PASSWORD
- PAYPAL_RECEIVER_EMAIL
- PAYPAL_TEST=False
- DEFAULT_STUDIO_EMAIL
- LOG_FOLDER

# Optional for dev

- USE_MAILCATCHER
- HEROKU
- DEBUG
- TRAVIS
